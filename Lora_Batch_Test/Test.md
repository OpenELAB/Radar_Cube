# 使用ESP32C3+lora测试
## 上电配置进入无线唤醒模式
**450ms左右**
```
通信模组的配置参数具有掉电保存功能，可以在第一次上电的时候进行配置，然后记录一个值在nvs里用来判断是否进行配置过，如果配置过则不再进行配置，否则进行配置。

第一次上电配置，普通模式的参数加上无线唤醒的参数

  "AT+MODE=0",    // 进入配置模式
  "AT+RFBR=6000", // 配置普通模式的空中传输速率6000bps
  "AT+LPWR=0",    // 关闭深度睡眠模式
  "AT+RFCH=18",   // 设置空中唤醒的频道433MHz
  "AT+PID=255",   // 设置PID255
  "AT+MAMP=2",    // 使能无线唤醒模式
  "AT+MLPWR=2",   // 使能无线唤醒模式2
  "AT+MID=17",    // 设置唤醒ID为17
  "AT+MODE=1"     // 退出配置模式
```

## 切换模式
### 普通模式
**134ms左右**
```
  AT+MODE=0
  AT+MAMP=0
  AT+MODE=1

```
### 无线唤醒模式
**131ms左右**
```
  AT+MODE=0
  AT+MAMP=2
  AT+MODE=1
```
